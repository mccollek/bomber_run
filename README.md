# Bomber Run

A top-down vertical scrolling bomber game inspired by 1942, built with Godot 4.6.

Pilot a bomber over endless ocean and islands. Shoot down enemy planes with your forward guns, bomb ships and ground defenses, and survive as long as you can against escalating waves of enemies.

## Controls

| Action | Key |
|--------|-----|
| Move | Arrow Keys / WASD |
| Fire Guns | Space / Z |
| Drop Bomb | X / Ctrl |

## Features

- Endless vertical scrolling with parallax ocean and island terrain
- Forward guns (vs air targets) and bombs (vs sea/ground targets)
- Enemy fighter planes, patrol boats, destroyers, and AA gun emplacements
- Wave-based spawning with progressive difficulty scaling
- Pixel art retro aesthetic (480x720 viewport)
- Chiptune SFX and music
- Persistent high score

## Running

Requires [Godot 4.6](https://godotengine.org/download/) or later.

```bash
godot --path bomber_run/
```

Or open the project in the Godot editor and press F5.
